1. Copy the code from the textbox below. To copy: tap once on the code, tap Select All then tap Copy.


2. Next, click Safari's Add Bookmark icon . Select “Add Bookmark” and without making any changes, bookmark this page by clicking “Save.”

4. Next, delete the URL shown and paste the code you copied in step 1: tap the URL shown in the second field, tap "Select All", tap "Paste".

5. Tap "Done" on the keyboard, which will return you to the list of bookmarks. Tap "Done" to finish editing bookmarks. Tap "Done" a third time to close the bookmarks list.
